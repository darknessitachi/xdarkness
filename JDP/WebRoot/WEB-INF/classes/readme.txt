欢迎使用ZCMS(泽元网站内容管理系统),开始使用之前请仔细阅读以下说明：

1、建议设置JAVA_OPTS为-Xms128m -Xmx256m或更高

2、如果使用DB2，则默认表空间页大小要求大于等于16K
    
3、War包第一次运行会出现初始化界面要求配置数据库连接，默认情况下配置完成后点击“下一步”
   会自动建表并导入初始数据，如果只需配置数据库连接，则选择“只配置数据库”；如果需要手
   动建表，则不选择“自动创建数据库表结构”，在初始化界面下载SQL文件手工执行后再点击“下
   一步”。 
		
4、如果使用自动创建的数据库，则网站登录用户名及密码都是admin，该用户具有最高权限，请
   及时修改该用户的密码。

5、登录后请根据实际情况修改"系统管理"-"配置项管理"中"前端应用地址项"的域名及端口部分。
   否则统计分析、投票、评论等功能接收不到数据。
